# Sample Job Description for Testing

## Senior Full Stack Developer

### Company
TechCorp Inc. | Remote | Full-time

### About the Role
We're looking for an experienced Senior Full Stack Developer to join our growing engineering team. You'll work on building scalable web applications that serve millions of users worldwide.

### Key Responsibilities
- Design and implement scalable backend services using Node.js and Express
- Develop responsive user interfaces using modern React patterns
- Collaborate with product and design teams to bring features to life
- Optimize application performance and ensure high availability
- Mentor junior developers and conduct code reviews
- Participate in architectural decisions and technical planning

### Required Qualifications
- 5+ years of professional software development experience
- Expert-level proficiency in JavaScript/TypeScript
- Strong experience with React.js and modern frontend frameworks
- Solid understanding of backend development with Node.js and Express
- Hands-on experience with MongoDB and/or PostgreSQL
- Experience with REST APIs and API design principles
- Proficiency with Docker and containerization
- Familiarity with AWS or similar cloud platforms
- Git version control expertise
- Excellent problem-solving skills

### Preferred Qualifications
- Experience with Next.js framework
- Knowledge of GraphQL
- Experience with Redis for caching
- Familiarity with microservices architecture
- CI/CD pipeline experience (GitHub Actions, Jenkins, etc.)
- Experience with test automation (Jest, Cypress)
- Kubernetes or container orchestration knowledge
- Experience mentoring developers
- Open source contributions

### Technical Stack
- Frontend: React.js, Next.js, Tailwind CSS, TypeScript
- Backend: Node.js, Express.js, REST APIs
- Database: MongoDB, PostgreSQL, Redis
- DevOps: Docker, Kubernetes, AWS
- Tools: Git, GitHub, VS Code

### Benefits
- Competitive salary: $150K-$200K
- Health insurance (medical, dental, vision)
- 401(k) matching (5%)
- Flexible work hours
- Remote work options
- Professional development budget
- Stock options

### Ideal Candidate Profile
- 5+ years full stack development experience
- Passionate about clean code and best practices
- Strong communication and collaboration skills
- Self-motivated and proactive problem-solver
- Comfortable with continuous learning
- Team player with leadership potential

### Interview Process
1. Initial screening call (30 minutes)
2. Technical assessment (2 hours)
3. Technical interview with senior developer (1 hour)
4. Team fit and discussion with hiring manager (1 hour)
5. Offer and negotiation
